---
title: 'List bots'
description: 'List all bots belonging to a user within the authenticated app.'
---

`GET /bot/api/v1/bots`

<ParamField header="Authorization" type="string" required>
  Bearer token for the app. Format: `Bearer <app-token>`
</ParamField>

## Query parameters

<ParamField query="user_id" type="string" required>
  Filter bots by user ID. Only bots owned by this user within the authenticated app are returned.
</ParamField>

## Response

An array of Bot objects, ordered by `created_at` descending.

<ResponseField name="[]" type="object[]">
  <Expandable title="Bot object properties" defaultOpen>
    <ResponseField name="id" type="string">
      Unique bot UUID.
    </ResponseField>

    <ResponseField name="app_id" type="string">
      ID of the app that owns this bot.
    </ResponseField>

    <ResponseField name="user_id" type="string">
      ID of the user who owns this bot.
    </ResponseField>

    <ResponseField name="name" type="string">
      Display name.
    </ResponseField>

    <ResponseField name="slug" type="string">
      URL-friendly identifier used in subdomain routing.
    </ResponseField>

    <ResponseField name="access_token" type="string">
      Bot-level access token for CLI commands and token authentication.
    </ResponseField>

    <ResponseField name="status" type="string">
      Current bot status: `created`, `starting`, `running`, `stopped`, or `error`.
    </ResponseField>

    <ResponseField name="config" type="object">
      OpenClaw configuration stored for this bot.
    </ResponseField>

    <ResponseField name="endpoint" type="string">
      Internal K8s service endpoint. Empty when the bot is not running.
    </ResponseField>

    <ResponseField name="expires_at" type="string">
      Expiry timestamp, or `null` if the bot never expires.
    </ResponseField>

    <ResponseField name="created_at" type="string">
      ISO 8601 creation timestamp.
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      ISO 8601 last-updated timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -s "https://api.clawhost.ai/bot/api/v1/bots?user_id=user-001" \
  -H "Authorization: Bearer $API_TOKEN"
```

</RequestExample>
